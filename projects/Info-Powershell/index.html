PowerShell

Incident Response Scenario

Scenario:

A user reports that their system is slow and unknown network activity is detected. You suspect a malicious reverse shell connection.

Question: Which command should you run first to detect suspicious connections?

Answer: netstat -ano

This command shows active connections and associated PIDs. You can match the PID with tasklist to identify the malicious process.
